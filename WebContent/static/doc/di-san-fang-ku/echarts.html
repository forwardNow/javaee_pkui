echarts

echarts（图表库）

1. 介绍

特性：

  • 官网：http://echarts.baidu.com/
  • 版本：3.7.2。（在PKUIv1.2.0中引入）
  • 运行平台：PC端、移动端
  • 浏览器兼容：IE8/9/10/11，Chrome，Firefox，Safari等（当前绝大部分浏览器）
  • 主题：已引入所有官方提供的主题JS

结构：

pkui/lib/component/echarts
    |-- 3.7.2.x
        |
        |-- theme
        |   |-- all.js          // 集成了所有主题（相当于合并所有主题脚本文件内容）
        |   |-- dark.js
        |   |-- infographic.js
        |   |-- macarons.js
        |   |-- roma.js
        |   |-- shine.js
        |   |-- vintage.js
        |
        |-- echarts.js        
        |-- echarts.min.js    // 使用压缩后的【完整版】
        |
        |-- wrapper.js  // 封装：处理依赖关系

2. 使用

参考：http://echarts.baidu.com/examples.html（官方示例）

seajs.use( [ "jquery", "echarts" ], function ( $ ) {
    $( document ).ready( function () {
        var
            theme = "dark",
            container = $( "#myEchart" ).get( 0 ),
            chart,
            option
        ;

        chart = echarts.init( container, theme);

        option = {
            // ...
        };

        chart.setOption( option );

    } );
} );
