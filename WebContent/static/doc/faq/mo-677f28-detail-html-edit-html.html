模板(detail.html/edit.html)

模板（detail.html、edit.html）

1. 介绍

  • 说明：PKUI采用 art-template 作为模板引擎
  • 下载：https://github.com/aui/art-template
  • 语法：http://aui.github.io/art-template/zh-cn/docs/syntax.html

2. 字典翻译

2.1 模板中的字典翻译

// model = { ..., deptLevel: 3, ...   }
<td>{{deptLevel | dicValue: "DIC_DEPT_LEVEL" }}</td>
<td>{{deptLevel | dicValue: "__CTX__/static/dic/DIC_DEPT_LEVEL.xml" }}</td>
==>
<td>分局</td>

注意：在PKUI.v1.0.10中才可在指定字典时使用“__CTX__”。

2.2 JavaScript中的字典翻译

PKUI.component.DataSource.getDicValue( "DIC_DEPT_LEVEL", "1" ); //=> "省厅"
PKUI.component.DataSource.getDicValue( "__CTX__/static/dic/DIC_DEPT_LEVEL.xml", "1" ); //=> "省厅"
PKUI.component.DataSource.getDicValue( PKUI.ctx + "/static/dic/DIC_DEPT_LEVEL.xml", "1" ); //=> "省厅"

3. 日期（日期时间）格式化

3.1 在模板中格式化

// model = { ..., addTime: 1505983719546（毫秒数）, ...   }
// model = { ..., addTime: "2017-09-21", ...   }
// model = { ..., addTime: "2017-09-21 16:50:43", ...   }
// model = { ..., addTime: "2017/09/21", ...   }
// model = { ..., addTime: "2017/09/21 16:50:43", ...   }
// model = { ..., addTime: "20170921", ...   }
// model = { ..., addTime: "20170921165043", ...   }
<td>{{addTime | dateFormat : "YYYY年MM月DD日 HH时mm分ss秒"}}</td> 
==> 
<td>2017年09月21日 16时50分43秒"</td>

可格式的日期时间值类型：

  • 毫秒数（数字类型）
  • YYYYMMDDHHmmss 和 YYYYMMDD
  • YYYY-MM-DD HH:mm:ss 和 YYYY-MM-DD
  • YYYY/MM/DD HH:mm:ss 和 YYYY/MM/DD

3.2 在JavaScript中的格式化

function formatter( date, format ) {
    var
        momentInstance
    ;
    if ( typeof date === "number" ) {
        momentInstance = moment( date );
    } else if ( typeof date === "string" ) {
        momentInstance = moment( date, [ "YYYYMMDDHHmmss", "YYYY-MM-DD HH:mm:ss", "YYYY/MM/DD HH:mm:ss" ] );
    }

    format = format || "YYYY-MM-DD";

    return momentInstance.format( format );
}
formatter( new Date().getTime(), "YYYY-MM-DD HH:mm:ss" );//=>"2017-09-22 13:57:52"
formatter( "20170922135752", "YYYY-MM-DD HH:mm:ss" );//=>"2017-09-22 13:57:52"
formatter( "2017/09/22 13:57:52", "YYYY-MM-DD HH:mm:ss" );//=>"2017-09-22 13:57:52"
