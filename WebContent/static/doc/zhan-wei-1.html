PKUI组件

PKUI组件

1. 组件封装方式

封装常用组件，以达到快捷使用的目的，如下：

// 传统使用方式1
<input type="text" name="addTime" id="addTime" />
<script>
    // 手动初始化（渲染）
    jQuery( "#addTime" ).datepicker( {
        "language": "zh-CN",
        "format": "yyyy-mm-dd"
    } );
</script>


// 传统使用方式2
<input type="text" name="addTime" id="addTime" 
       data-provide="datepicker"
       data-language="zh-CN"
       data-format="yyyy-mm-dd" />
// 自动初始化（渲染）
// 通过以下方式自动渲染，该代码内置于此插件
$(document).on(
    'focus.datepicker.data-api click.datepicker.data-api',
    '[data-provide="datepicker"]',
    function(e){
        var $this = $(this);
        if ($this.data('datepicker'))
            return;
        e.preventDefault();
        // component click requires us to explicitly show it
        datepickerPlugin.call($this, 'show');
    }
);

// PKUI组件使用方式
<input type="text" name="addTime" id="addTime" 
       data-pkui-component="datepicker"
       data-pkui-component-options='{
          "language": "zh-CN",
          "format": "yyyy-mm-dd"
       }' />
// 自动初始化（渲染）

2. PKUI组件结构

<div ... 
    data-pkui-component="datepicker"
    data-pkui-component-options='{
       "language": "zh-CN",
       "format": "yyyy-mm-dd"
    }' ></div>

data-pkui-component ：指定组件的名称。

data-pkui-component-options：指定组件的参数；对于封装的第三方组件，也是在这里传入参数的。

3. PKUI组件自动渲染机制

1、当 DOM树构建完毕后 或 $jquery.html( ... ) 方法被执行后；

2、查找使用PKUI组件的DOM节点：$( "[data-pkui-component]" ).not( "[isrendered]" ).not( "[notrecognized='not reg']" )；

3、根据 data-pkui-component 指定的组件名，传入data-pkui-component-options指定的参数，然后进行渲染；

4、渲染完毕添加自定义属性 isrendered 标志已被渲染。（也就是说，有isrendered属性的元素不会进行渲染）

4. PKUI组件（模块别名）

  • bootgrid：数据表格（datagrid），参看 app/desktop/tpl/system/dept/list.html。
  • datagrid-delete：配合数据表格使用的删除按钮，参看 app/desktop/tpl/system/dept/list.html。
  • drawer：将元素以抽屉的形式展开，用于显示详情页和编辑页，参看 app/desktop/tpl/system/dept/list.html。
  • form：在表单（<form>）上使用，让表单以Ajax的方式提交，参看 app/desktop/tpl/system/dept/edit.html。
  • validator：表单域验证，通常配合form组件使用，参看 app/desktop/tpl/system/dept/edit.html。
  • chosen：下拉选，可配合字典（如DIC_GENDER.xml）使用，参看 chosen。
  • webuploader：上传组件，参看 app/desktop/tpl/system/dic/list/import.html。
  • menutree：菜单树，参看 app/desktop/tpl/desktop/appWindow.html。
  • Intro：引导组件，参看 app/desktop/tpl/system/menu/list.html。
  • loadContent：在指定元素内载入内容，参看 app/desktop/tpl/system/index.html。
  • datepicker：日期控件，参看 app/desktop/tpl/system/log/list.html。
  • datetimepicker：日期时间控件，参看 app/desktop/tpl/system/log/list.html。
  • textMagnifier：文本放大器，参看 app/desktop/tpl/system/user/add.html。
