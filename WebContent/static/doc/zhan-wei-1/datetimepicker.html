date(time)picker

datepicker & datetimepicker

由于前期只考虑到日期（如 2017-09-11）的录入，

引入了 datepicker（https://github.com/uxsolutions/bootstrap-datepicker）。

而在实际开发过程中，有些场景需要录入日期和时间（如 2017-09-11 10:02:47），

因此又引入了 datetimepicker（https://github.com/xdan/datetimepicker）。

datepicker 的使用

示例：

<input type="text" name="logTime" placeholder="请选择"
    data-pkui-component="datepicker"
    data-pkui-component-options='{
        "format": "yyyy-mm-dd",
        "autoclose": false
    }'/>

参数的设置：

参看 https://bootstrap-datepicker.readthedocs.io/en/stable/options.html

注意：

PKUIv1.0.9之前的版本，desktop的样式与datepicker的样式冲突，致使通过伪元素:before和:after构造尖角失败，而遮住了日期选择面板，造成其不可用。（此bug已在PKUIv1.0.9中修复）

datetimepicker 的使用

示例：

// 日期时间
<input type="text" name="logTime" placeholder="请选择"
    data-pkui-component="datetimepicker"
    data-pkui-component-options='{
        "format": "Y-m-d H:i:s",
        "formatTime": "H:i:s",
        "formatDate": "Y-m-d",
        "dayOfWeekStart" : 1,
        "datepicker": true,
        "timepicker": true
    }'/>

// 日期        
<input type="text" name="logTime" placeholder="请选择"
    data-pkui-component="datetimepicker"
    data-pkui-component-options='{
        "format": "Y-m-d",
        "formatDate": "Y-m-d",
        "datepicker": true,
        "timepicker": false
    }'/>

// 时间
<input type="text" name="logTime" placeholder="请选择"
    data-pkui-component="datetimepicker"
    data-pkui-component-options='{
        "format": "H:i:s",
        "formatTime": "H:i:s",
        "datepicker": false,
        "timepicker": true
    }'/>

参数的设置：

参看 https://xdsoft.net/jqplugins/datetimepicker/ （Full options list）

说明：

由于可以通过 属性设置时间选择面板的现实与否，所有其可以当成日期选择器来使用，推荐使用 datetimepicker而不是datepicker。
