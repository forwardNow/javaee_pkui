应用于四川项目的记录

访问：http://localhost:8081/rkxtbasic/pkuiLogin

1. 获取 pkui包 及 desktop包

下载地址：https://github.com/forwardNow/javaee_pkui/releases

下载最新的版本（50M左右），解压后，目录如下：

javaee_pkui-1.x.x
|-- build
|-- doc
|-- resource
|-- src
|-- WebContent
    |-- WEB-INF
    |-- static
        |-- pkui        // 拷贝此目录
        |-- desktop     // 拷贝此目录

将 pkui 和 desktop 拷贝你的项目的 static 目录。
