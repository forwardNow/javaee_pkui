功能介绍

1. 查看

操作：单击菜单项，在右侧查看该菜单的详细内容。

2. 新建（）

1）操作：【新建顶级菜单】不选中任何菜单，直接点击“新建”按钮后，在编辑界面进行编辑并保存。

2）操作：【新建子菜单】选中某菜单后，点击“新建”按钮后，在编辑界面进行编辑并保存。

3）注意：新建的菜单会被标记为橙色；如果存在未保存的新建的菜单，则不允许新建第二个菜单。

3. 编辑（）

1）操作：选中菜单后，点击“编辑”按钮，在右侧编辑界面进行修改。

2）保存：编辑完毕后，请点击编辑界面底部的保存按钮进行保存，保存成功后会自动刷新菜单树。

4. 删除（）

操作：选中菜单后，点击“删除”按钮，会弹出的警告框，点击“删除”进行删除，点击“取消”则不删除。

5. 保存（）

1）操作：对菜单项进行拖拽排序，排序完毕后点击“保存”，保存菜单树结构。

2）说明：被影响的菜单项会被标记为紫色。

6. 刷新（）

操作：点击“刷新”，重新载入目录树。
